# W3RP Tool

利用webpack基于React/React-Router/Redux搭建的工程化环境

## How to start

```bash
npm install
npm run start
```
            
## How to build

```bash
npm run build
```

## Some dependencies

>一套优雅的UI组件库[v7ui](https://www.npmjs.com/package/v7ui)
